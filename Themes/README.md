# Imdad – Military Edition v1.0
نظام إمداد راجمات الصواريخ – النسخة العسكرية الرسمية.

## نظرة عامة
هذا المستودع يحتوي على الهيكل الأساسي لنظام الإمداد العسكري، مع قابلية التوسّع وإضافة الملفات لاحقًا.  
تم تصميم النظام ليكون مرنًا، قابلًا للتطوير، وملائمًا للاستخدام العسكري الاحترافي.

## محتويات المشروع
- واجهة هجينة متقدمة (App)
- نظام وحدات عسكرية (Modules)
- نظام ثيمات قابل للتخصيص (Themes)
- أدوات مساعدة للنظام (Tools)
- وثائق وشروحات كاملة (Docs)
- قاعدة بيانات SQLite

## حالة المشروع
تم إنشاء الإصدار الأول الرسمي v1.0  
وسيتم إضافة الملفات التشغيلية لاحقًا حسب الحاجة.

## هيكلة المجلدات
- **App** – ملفات الواجهة الأمامية للنظام  
- **Docs** – الوثائق والشروحات  
- **Modules** – الوحدات العسكرية (أسلحة – مركبات – أقسام)  
- **Themes** – الثيمات والأنماط  
- **Tools** – الأدوات المساعدة  

## ملاحظات
المجلدات قد لا تظهر في واجهة GitHub إذا كانت فارغة، وتظهر تلقائيًا عند إضافة أي ملف داخلها.
