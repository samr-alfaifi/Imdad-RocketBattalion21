# Architecture – هيكلة النظام

هذا المستند يشرح الهيكلة العامة لنظام الإمداد العسكري.

## المكونات الأساسية
- App: واجهة المستخدم
- Modules: الوحدات العسكرية
- Themes: الثيمات والأنماط
- Tools: الأدوات المساعدة
- Docs: الوثائق والشروحات
- Database: قاعدة البيانات

## طريقة العمل
النظام يعتمد على هيكلة مرنة تسمح بإضافة وحدات جديدة بدون تعديل على النظام الأساسي.
